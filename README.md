# doom.lat

Minimalist portfolio website with smooth animations and integrated music player.

## Features

-  Clean black & white gradient design
-  Smooth opening animations with Framer Motion
-  Custom cursor with interactive effects
-  Built-in MP3 player
-  Built with Next.js 14, TypeScript, and Tailwind CSS

## Local Development

1. Clone this repository
2. Install dependencies: `npm install`
3. Add your MP3 file to `public/music/track.mp3`
4. Run dev server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/doom.lat)

Or manually:
1. Push this repository to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Vercel auto-detects Next.js and deploys

## Adding Your Music

Upload your MP3 file directly on GitHub:
1. Navigate to `public/music/`
2. Click "Add file" → "Upload files"
3. Upload your track as `track.mp3`
4. Commit changes
